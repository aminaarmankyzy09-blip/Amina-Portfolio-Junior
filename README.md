🛠 Технические особенности проекта:

Объектно-ориентированное программирование (ООП): Вся логика работы с пользователем и данными инкапсулирована внутри класса JuniorPortfolio.

Разделение логики и контента: Вся информация о блоках портфолио вынесена в отдельный структурированный JSON-файл (content.json). Программа динамически считывает данные при старте.

Безопасная обработка ошибок: Файловые операции защищены блоками try-except, предотвращая падение программы при отсутствии файлов.

Защита ввода через регулярные выражения (Regex): Для навигации по меню используется проверка пользовательского ввода с помощью библиотеки re. Ввод букв или некорректных цифр безопасно обрабатывается программой.

Интерфейс программы: Оформлено аккуратное и читаемое меню с использованием символьных рамок-разделителей в консоли.

📁 Структура репозитория

main.py — Основной файл программы, содержащий класс JuniorPortfolio и цикл работы меню.

content.json — JSON-файл с текстами всех 8 разделов портфолио.

homeworks/ — Папка с практическими работами/домашними заданиями в процессе обучения.

README.md — Описание и техническая документация к проекту.

💻 Инструкция по запуску

Для запуска консольного портфолио выполните в терминале команду:
